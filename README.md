A complete, content-rich personal website celebrating Web 1.0, GeoCities aesthetics, Linux culture, and digital freedom.

## 🚀 Quick Start

### View Locally
```bash
# Navigate to project
cd nemooWeb

# Start local server
python3 -m http.server 8000

# Open browser
open http://localhost:8000
```

### Deploy
See `docs/INSTALL.md` for deployment instructions.

## 🎨 Design Philosophy

- **GeoCities Aesthetic** - Bright colors, tiled backgrounds, beveled buttons
- **No Modern Frameworks** - Pure HTML, CSS, and vanilla JavaScript
- **Privacy First** - No tracking, no cookies, no external dependencies
- **Modular Structure** - Easy to customize and extend
- **Content Rich** - Real writing, not placeholder text
