guestbook

A browser-stored guestbook that keeps the old-web feeling without inventing a fake backend. It works entirely in localStorage and is honest about that.

  • write and store entries in the current browser
  • seed with sample notes so the page never starts dead-empty
  • reset or clear entries without leaving the page

build principles

  • Static by default. Dynamic only when the interaction actually needs state or persistence.
  • No fake completeness. Missing ideas stay described, not linked.
  • Visual identity matters, but coherence matters more than stuffing every page with effects.
  • The site should be understandable as files, not as an opaque toolchain.
