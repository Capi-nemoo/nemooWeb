What I Build

These are my digital experiments—tools, toys, and projects built in the spirit of the old internet. They're not polished products. They're not monetized platforms. They're just things I made because I wanted to make them.

Some are useful. Some are silly. All are built with the aesthetic and philosophy of Web 1.0.

Linux ID Generator 2025

Status: Active • Type: Web Tool • Year: 2025

A retro-styled Linux user ID card generator. Create your own digital identity card with your username, distro, favorite shell, and more. Inspired by the ID cards and badges of early internet forums.

Features:

  • Customizable user information
  • Multiple retro design templates
  • Export as image
  • Authentic 1990s aesthetic
  • No tracking, no data collection

Interactive Terminal Emulator

Status: Active • Type: Interactive Toy • Year: 2025

A fake terminal that actually works. Type commands, explore the filesystem (simulated), and experience the joy of the command line without the risk of breaking anything.

Available Commands:

  • help - Show all commands
  • ls - List files
  • cat [file] - Read files
  • cowsay [text] - ASCII cow art
  • hack - Fake hacking sequence
  • And more...

Project Philosophy

All projects on this site follow these principles:

  • No Tracking - Your data stays on your device
  • No Ads - These are passion projects, not profit centers
  • Open Source Spirit - Learn from the code, modify it, share it
  • Retro Aesthetic - Embrace the look and feel of Web 1.0
  • Functional Fun - Useful tools that are also enjoyable to use

Want to Contribute?

These projects are built in the open. If you have ideas, improvements, or want to build something similar, go for it. The old internet was built by people sharing and remixing each other's work.

View the source code. Learn from it. Make it better. Build your own version. That's the spirit of the web.
